# LaTeX as a Service

**See this app live on the web:** https://laas.vercel.app

## Getting Started

**Clone the repo:**

```sh
git clone git@github.com:jozsefsallai/laas.git
cd laas
```

**Install the dependencies:**

```sh
npm i -g yarn
yarn
```

**Run the dev server:**

```sh
yarn dev
```

**Make sure your changes pass linting:**

```sh
yarn lint
```
